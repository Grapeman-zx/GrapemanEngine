GrapmanEngine
GrapmanEngine 是一款基于 C++20 和 DirectX 12 API 构建的现代图形渲染引擎。本项目旨在实现一个高性能、可扩展的底层渲染架构，专注于 D3D12 的深度特性利用与现代图形技术实验。

核心特性
现代 D3D12 架构：深度封装 ID3D12Device、ID3D12CommandQueue 等核心组件。

资源管理系统：支持自动化的显存资源生命周期管理。

技术栈
Language: C++ 20

Graphics API: DirectX 12 (Feature Level 11_0+)

Build System: Visual Studio 2026 (MSBuild)

SDK: Windows 10/11 SDK (10.0.19041.0+)
